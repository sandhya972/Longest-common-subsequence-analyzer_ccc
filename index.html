About LCS

The core algorithm behind diff tools and plagiarism detection.

🧠

What is LCS?

The Longest Common Subsequence is the longest sequence that can be derived from two input strings by deleting zero or more characters without changing the order.

⚙️

Dynamic Programming

The algorithm uses a 2D matrix to store solutions to subproblems. This ensures that we only compute each state once, resulting in O(M×N) efficiency.

🚀

Applications

Used in git diff, bioinformatics for DNA sequence alignment, and data compression algorithms like GZIP.
